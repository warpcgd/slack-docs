# 覆盖组件样式

由于业务的个性化需求，我们经常会遇到需要覆盖组件样式的情况，这里举个简单的例子。

  • 将头像框加上外层 border
<template>
  <div :class="$style.container">
    <Avatar
      :class="$style.avatar"
      :src="'https://o-id.ihago.net/blob/v1/ali/id/uurl/1617875956701102.jpeg'"
    >
    </Avatar>
  </div>
</template>

<script lang="ts">
import { Vue, Component } from 'vue-property-decorator'
import Avatar from '@yy/slack-ui/dist/lib/avatar'
import '@yy/slack-ui/dist/lib/avatar/index.css'
@Component({
  components: {
    Avatar
  }
})
export default class test extends Vue {}
</script>
<style lang="scss" module>
.avatar {
  width: 100px;
  height: 100px;
  img {
    width: 100%;
  }
}
/* css-module 覆盖 */
.container {
  :global {
    .slack__avatar__container {
      border-radius: 100%;
      border: 3px solid #333;
    }
  }
}
</style>
<style lang="scss" scoped>
/* css style 覆盖 */
.slack__avatar__container {
  border: 3px solid #cc0000;
}
</style>
code-show ▾

方法

  • css-module 覆盖: slack-ui 组件类名会被 CSS Modules 转化，所以要覆盖的类名 .slack__avatar__container 必须放到 :global 中。
  • 直接 style 覆盖: 可以直接使用sfc的 style 覆盖
